Important — password security

Passwords are stored as reversible base64 encoding to allow admin recovery. When assisting a member, provide their password securely (phone call or in-person if possible) and ask them to change it on next login. This system is for BETA use only — a full hashed authentication system will be implemented before full deployment.

Recovery Requests
Password reset and member number recovery requests submitted by members from the Independence Vault login screen. Each request includes the member's email — reply directly to assist them.
How to respond
Password reset: Verify identity against the member register, then reply to their email with a temporary password. Ask them to change it on first login.

Member number recovery: Look up their name/email in the COG$ member database Excel file, find their 16-digit number, and reply with it securely.
